# FrequencyGenerator

</div>

A simple, clean web app to generate audio frequencies from 1 Hz to 22 kHz.

## Features

- **Tone Generation**: Generate sine, square, sawtooth, and triangle waves.
- **Channel Control**: Pan the audio to the left, right, or both channels.
- **Volume Control**: Adjust the volume of the audio.
